# Python Device Authorization Sample

This sample demonstrates how to perform device authorization with Konnektr's OAuth2 endpoint and query the API using the obtained access token.

## Prerequisites

- Python 3.7+
- `requests` library (install with `pip install -r requirements.txt`)

## How to Run

1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Run the script:
   ```sh
   python device_auth_sample.py
   ```
3. Follow the instructions in the terminal to complete device authentication in your browser.

## What it does

- Initiates device authorization with Konnektr
- Polls for an access token
- Uses the token to query the API for digital twins
